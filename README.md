## ⚠️ Cadre légal — à lire absolument

Les exercices de pentest de ce module portent sur une application **dont l'entreprise KodeSaaS est propriétaire et qui autorise explicitement ces tests dans un cadre pédagogique**.

> **Un test d'intrusion sur un système que vous ne possédez pas, sans autorisation écrite, est un DÉLIT** (articles 323-1 et suivants du Code pénal). Ne reproduisez **jamais** ces techniques sur un site tiers sans autorisation. Pas d'exception.
